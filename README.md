## Tech Stack

- react
- vite
- Redux Toolkit
- RTK Query
- json-server

## Startup

### To start the server

- copy files locally and cd to the directory
- npm install
- npm run dev
- in a second terminal, start json-server:
- npx json-server --watch --port 3500 data/db.json
- in a browser connect to localhost:5173
